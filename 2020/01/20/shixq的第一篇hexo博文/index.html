4. 部署到github上

新建一个git仓库，地址填为应户名.github.io

模板 用途
index 首页
post 文章
page 分页
archive 归档
category 分类归档
tag 标签归档

6. KaTeX

数学公式

我的配置完之后不能使用，换成MathJax

在文字处理领域，早就有非常成熟的排版系统LaTeX，它完全使用文本描述排版复杂的数学公式、表格、化学分子式等，甚至可以排五线谱、棋谱、电路图等。

博客网页上的数学公式，只需要一个LaTex的子集来渲染数学公式。KaTeX就是这样一个支持HTML的轻量级的数学公式引擎，它由Khan Academy开发，使用起来也非常简单。

内嵌的Katex效果$E=mc^2$就是这样

$E=mc^2$

修改主题配置文件butterfly.yml,启用katex(enable: true) , 在使用时Markdown Front-matter 上加上 katex: true

卸载之前的 hexo 的 markdown 渲染器以及 hexo-math，然后安装新的 hexo-renderer-markdown-it-plus

卸载hexo-math会删掉宠物插件live2d

hexo-renderer-markdown-it-plus已经无人持续维护，所以使用 @upupming/hexo-renderer-markdown-it-plus。 这份 fork 的代码使用了 @neilsustc/markdown-it-katex同时它也是 VSCode 的插件 Markdown All in One 所使用的，可以获得最新的 KaTex 功能例如 \tag{}。

还可以通过 @neilsustc/markdown-it-katex控制 KaTeX 的设置，所有可配置的选项参见 https://katex.org/docs/options.html。 比如想要禁用掉 KaTeX 在命令行上输出的冗长的警告信息，可以在根目录的 _config.yml 中使用下面的配置将 strict 设置为 false:

7. tag-hide

Inline
inline 在文本里面添加按钮隐藏内容，只限文字

( content 不能包含英文逗号，可用&sbquo;来代表逗号)

content: 文本内容

display: 按钮显示的文字 (可选)

bg: 按钮的背景颜色 (可选)

color: 按钮文字的颜色 (可选)
